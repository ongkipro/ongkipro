## Philosophy

- **Terminal-first** — CLI over GUI. Speed through muscle memory.
- **Ship fast, iterate** — Commit early, deploy often, fix in production when safe.
- **AI as force multiplier** — Not a replacement. Prompt engineering is a core skill.
- **Open by default** — Reference docs, tools, and skills are public where possible.
- **No "Buy Now" CTAs in API docs** — Documentation is declarative. Marketing has its place.
